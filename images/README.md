# Carpeta /images

Aquí van las imágenes del blog técnico. Los archivos referenciados en el HTML son:

| Archivo                   | Alt text en HTML                        | Tamaño sugerido |
| ------------------------- | --------------------------------------- | --------------- |
| `javascript-moderno.webp` | Pantalla con código JavaScript moderno  | 800×450 px      |
| `git-workflow.webp`       | Diagrama de flujo de ramas en Git       | 800×450 px      |
| `solid-principles.webp`   | Diagrama con los cinco principios SOLID | 800×450 px      |

## Cómo obtener las imágenes

Puedes:

1. Crear tus propias imágenes con diseño relacionado al tema.
2. Usar imágenes libres de derechos (ej. Unsplash, Pexels) y renombrarlas.
3. Convertirlas a formato `.webp` para optimizar el rendimiento web.

> **Tip**: Si no tienes imágenes locales, puedes temporalmente reemplazar el atributo `src`
> con URLs externas de Unsplash. El HTML seguirá siendo válido en W3C.
